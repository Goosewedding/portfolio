Usage

How to use icons in HTML

Inline SVG (recommended)

Paste the SVG code directly into your HTML. The icon inherits color from its parent element via currentColor.

<!-- Icon inherits color from parent --> <span style="color: white"> <svg width="16" height="12" viewBox="0 0 14 12" fill="none"> <path d="M14 12H0V0H3.5814L4.55814 1H14V12Z" fill="currentColor"/> </svg> </span>

As an <img> tag

Simpler to write, but cannot be recolored via CSS — the icon will always render in its native color.

<img src="icons/folder.svg" width="24" alt="Folder">

Sizing in CSS

/* Control size via CSS instead of HTML attributes */ .icon { width: 1em; /* scales with font-size of parent */ height: 1em; display: inline-block; vertical-align: middle; }
