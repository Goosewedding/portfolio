Nav Button

.btn-nav

Default state

Transparent background · white text · designed for dark surfaces

Hover state

White fill (#FFFFFF) · black text · no border, no radius

Interactive (hover me)

Specs

CSS class .btn-nav
Font TT Mussels Trial · Bold (700)
Font size 16px · var(--text-base)
Padding 4px top/bottom · 8px left/right
Gap 8px (between icon and label)
Border radius 0 — sharp corners
Default bg transparent
Default text #FFFFFF
Hover bg #FFFFFF
Hover text #000000
Transition 150ms ease (background-color + color)
Surface Dark background only

CSS

/* Nav Button — default (on dark surface) */ .btn-nav { display: inline-flex; align-items: center; justify-content: center; gap: 8px; font-family: var(--font-display); font-size: var(--text-base); font-weight: var(--weight-bold); line-height: 1; padding: 4px 8px; background-color: transparent; color: var(--color-white); border: none; border-radius: var(--radius-none); transition: background-color 150ms ease, color 150ms ease; } /* Hover state */ .btn-nav:hover { background-color: var(--color-white); color: var(--color-black); }

HTML

<!-- As a button --> <button class="btn-nav">Index</button> <!-- As a link --> <a href="/work" class="btn-nav">Work</a>
